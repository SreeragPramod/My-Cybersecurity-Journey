# Python Security Tools

Daily Python scripts written while learning cybersecurity fundamentals.
Each script targets a specific security concept — reconnaissance,
authentication, encoding, log analysis, or network filtering.

| Script | Concept | Skill Demonstrated |
|--------|---------|---------------------|
| day01_ip_lookup.py | DNS resolution | Reconnaissance |
| day02_password_checker.py | Password strength | Authentication security |
| day03_pin_cracker.py | Brute force | Attack mechanics / defense awareness |
| day04_password_hasher.py | Hashing | Cryptographic basics |
| day05_firewall_simulator.py | Packet filtering | Network security concepts |
| day06_log_analyzer.py | Log parsing | Threat detection |
| day07_base64_encoder.py | Encoding | Data obfuscation techniques |
| day08_port_scanner.py | Port scanning | Network mapping|

Each script includes a docstring explaining its real-world security relevance.
